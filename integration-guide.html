Futuristic Space Theme

A comprehensive design system inspired by NASA dashboards and space visualization interfaces. Complete with glassmorphism, neon accents, and immersive animations.

Quick Start Integration

To apply this theme to your existing project, follow these steps:

<!-- Add to your HTML head section -->
<link rel="stylesheet" href="theme.css">

Apply Theme Classes

  • Wrap content in <div class="container">
  • Use .card or .panel for glassmorphism containers
  • Add .fade-in for entrance animations
  • Use .pulse-glow for active elements
  • Apply .btn or .btn-primary to buttons

CSS Variables Reference

All theme values are available as CSS custom properties:

/* Colors */
var(--color-bg)
var(--color-accent)
var(--color-text)
var(--color-muted)

/* Spacing */

var(--spacing-sm) /* 16px */
var(--spacing-md) /* 24px */
var(--spacing-lg) /* 32px */

/* Fonts */

var(--font-body) /* Space Grotesk */
